# Bryan Leung — React Personal Website

React + Vite portfolio with fancy effects (scroll reveal, tilt cards, magnetic CTA, animated starfield background).

## Run locally

```bash
npm install
npm run dev
```

## Update your LinkedIn

In `src/App.jsx`, replace:

- `https://www.linkedin.com/in/YOUR_LINKEDIN_HANDLE`
- `@your-profile`

## Customize

- **Text/content**: `src/App.jsx`
- **Colors/styles**: `src/styles.css` (see `:root` variables)
- **Effects/animations**: `src/App.jsx` (the `useFancyEffects` hook)
